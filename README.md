## 🧠 Core Concepts Mastered
This repository documents my exploration into several advanced recursive strategies:

* **Backtracking**: Exploring all possible paths in a search space and abandoning candidates ("pruning") as soon as it's determined they cannot lead to a valid solution.
* **Divide & Conquer**: Breaking down complex tiling and grid problems into smaller, manageable recursive sub-problems.
* **State Space Trees**: Visualizing recursive calls as trees to understand time complexity and redundancy.
